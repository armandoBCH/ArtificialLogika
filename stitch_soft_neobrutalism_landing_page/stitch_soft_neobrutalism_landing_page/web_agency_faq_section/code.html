¿Cuánto tarda en estar lista mi web?

add
Depende de la complejidad del proyecto. Para una Landing Page estándar, el tiempo estimado es de 1 a 2 semanas una vez que tenemos todo el contenido. Para sitios corporativos o e-commerce, solemos tardar entre 3 y 5 semanas. Siempre trabajamos con un cronograma claro desde el día uno.

¿Qué incluye el precio?

add
Incluye diseño UX/UI personalizado, desarrollo responsive (adaptable a celulares), configuración básica de SEO, integración con redes sociales y un formulario de contacto. Además, te entregamos un video tutorial para que puedas autogestionar cambios simples de texto e imágenes. El hosting y dominio no están incluidos, pero te asesoramos para comprarlos.
